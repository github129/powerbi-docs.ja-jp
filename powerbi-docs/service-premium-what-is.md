# <a name="what-is-power-bi-premium"></a>Power BI Premium とは

Power BI Premium は、組織の Power BI サービスを実行する専用および強化されたリソースを提供します。 例:

- スケールとパフォーマンスの向上
- 容量によってライセンスの柔軟性
- セルフ サービスとエンタープライズ BI を統合します。
- Power BI Report Server によるオンプレミスで BI を拡張します。
- データの保存場所のリージョン (Multi-geo) 別のサポート
- ユーザーごとのライセンスを購入しなくても、だれとでもデータを共有します。

この記事では、実際に Power BI Premium とのすべての機能についての詳細を提供するものではありません、サーフェスを当てはめますだけです。 必要に応じて、詳細な情報をその他の記事へのリンクが提供されます。
