# <a name="use-cross-report-drillthrough-in-power-bi-desktop"></a>Power BI Desktop でクロス レポートのドリルスルーを使用します。

Power BI Desktop でクロス レポートのドリルスルー機能を使用するコンテキストからにジャンプできます 1 つのレポートから別のレポート。 これは、同じワークスペースまたは Microsoft Power BI サービスでのアプリ内では、レポートを限り true。 クロス レポートのドリルスルーを使用して、コンテンツに関連付けられている 2 つ以上のレポートの接続間レポート接続と共にフィルター コンテキストを渡します。 この記事では、Power BI レポートでは、クロス レポートのドリルスルーを設定する方法と、ユーザーが自分間レポートのドリルスルーを使用するときに発生するについて説明します。

![Power BI Desktop のスクリーン ショットのドリルスルーのオプション](media/desktop-cross-report-drill-through/cross-report-drill-through-01.png)

次の定義を理解しておく必要は、始める前に設定および使用するクロス-レポートのドリルスルー。

* **ソース視覚エフェクト:** このビジュアルは、ビジュアルのコンテキスト メニューを使用して、ドリルスルー アクションを呼び出します。
* **ソース レポート:** このレポートは、クロス レポートのドリルスルーのソースのビジュアルが含まれています。
* **ターゲット ページ:** ユーザーがドリルスルー アクションを開始した後にアクセスしたページです。
* **ターゲット レポート:** このレポートは、クロス レポート ドリルスルー ターゲット ページが含まれています。

## <a name="enable-cross-report-drillthrough"></a>クロス レポートのドリルスルーを有効にします。

間レポートのドリルスルーの対象となるレポートを有効にするには、そのレポートでの機能を有効にする必要があります、**オプション**ウィンドウ。 移動して**ファイル** > **オプションと設定** > **オプション**、順に移動**設定がレポート**近く、左側のページの下部にあります。

選択、**他のレポートからドリルスルー ターゲットを使用するには、このレポートでビジュアルを許可する**チェック ボックスを次の図のようにします。

![オプションのスクリーン ショット] ウィンドウ [強調表示されているレポートの設定](media/desktop-cross-report-drill-through/cross-report-drill-through-02.png)

## <a name="set-up-cross-report-drillthrough"></a>クロス レポート ドリルスルーを設定します。

クロス レポートのドリルスルーの設定は、レポート内でのドリルスルーの設定に似ています。 ドリル スルーがターゲット ページで、有効になり他のビジュアルのドリルスルーが有効なページを対象にします。 1 つのレポート内のドリルスルーを作成する手順については、次を参照してください。 [Power BI Desktop でドリルスルーを使用して](desktop-drillthrough.md)します。

セットアップ プロセスを開始するには、いくつかの初期手順を実行する必要があります。

* ワークスペースまたはアプリで他のレポートからアクセスできる、ドリルスルー ターゲット ページを設定します。
* 独自のレポートの外部からのドリルスルー ページを表示するレポートを許可します。

ドリルスルーのオプションを検索、**フィールド**のセクション、**視覚化**ウィンドウで、次の図のようにします。

![スクリーン ショットの視覚エフェクト ウィンドウで、ドリルスルー オプションが強調表示されています。](media/desktop-cross-report-drill-through/cross-report-drill-through-03.png)

* 渡すフィールドは、両方のデータ モデル内に存在します。
* フィールドの名前と所属する、テーブルの名前は同じです (文字列も一致する必要があり、小文字が区別されます)。

セットアップを開始するには、ターゲット ページを準備する必要があります。 Power BI Desktop でのページに移動し、ことを確認します、**クロス レポート**に設定されているドリル スルー切り替え**で**します。

![ドリル スルーが強調表示されていると、Power BI サービス内のソース レポートのスクリーン ショット](media/desktop-cross-report-drill-through/cross-report-drill-through-01.png)
